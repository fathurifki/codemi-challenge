### How To Run 
```
1. clone the Repo 1st
2. npm install
3. npm start
```

### Folder Structure

```
A. assets : Folder for Styling For Dashboard
B. components : Folder for Components
  1. Card
  2. Chart // 
  3. Grid
  4. layouts // custom route for layout dashboard
  5. Navbar
  6. Topbar
C. containers // View components
D. images

```

 You can check demo [here](https://codemi-challenge.vercel.app/dashboard/home)
